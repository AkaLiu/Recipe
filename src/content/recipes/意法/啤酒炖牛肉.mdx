## 食材

| 用量 | 食材 |
|------|------|
| 1kg | 牛肩肉 |
| 1L | 棕色啤酒 |
| 1个 | 洋葱 |
| 3根 | 胡萝卜 |
| 50g | 黄油 |
| 适量 | 盐和胡椒 |
| 1束 | 香草束 |
| 1kg | 土豆 |

## 步骤

1. 将牛肉切成3厘米的方块。
2. 在平底锅中用融化的黄油高火煎牛肉块。
3. 加入啤酒、洋葱和粗切的胡萝卜。
4. 小火慢炖4到5小时。肉应该软烂，酱汁浓稠。
5. 土豆去皮切成四等份，撒盐、胡椒，加入一些香草和橄榄油。
6. 放入烤箱210°C烤45分钟。

## 小贴士

- 炖煮时锅内应该保持微沸状态。
- 这道菜是比利时和法国北部的传统菜。
- 搭配薯条或法棍面包食用。
