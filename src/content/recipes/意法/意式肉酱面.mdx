## 食材 | Ingredients

| 用量 | 食材 |
|------|------|
| 750g | 牛肉末 |
| 250g | 猪肉末 |
| 1L | 番茄酱 |
| 1罐 | 番茄碎 |
| 3根 | 胡萝卜 |
| 3根 | 芹菜 |
| 2个 | 洋葱 |
| 500g | 意面（Tagliatelle） |
| 100g | 帕玛森芝士 |

## 步骤 | Steps

1. 将蔬菜切成小丁（mirepoix）。
2. 在橄榄油中大火翻炒蔬菜。
3. 当洋葱变透明、蔬菜软化后，加入肉末。不断翻散。
4. 当肉不再出水时，加入番茄碎和番茄酱。
5. 至少炖煮4小时。
6. 在盐水中煮意面（9克盐/升）。

## 小贴士 | Tips

- 搭配帕玛森芝士食用。
- 这是博洛尼亚肉酱面的经典做法。
- 肉酱可以冷冻保存。
