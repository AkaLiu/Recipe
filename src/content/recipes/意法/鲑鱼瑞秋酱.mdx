## 食材 | Ingredients

| 用量 | 食材 |
|------|------|
| 5块 x 200g | 鲑鱼排 |
| 4个 | 彩色甜椒 |
| 1头 | 大蒜 |
| 2束 | 香菜 |
| 3个 | 柠檬 |
| 20cl | 橄榄油 |
| 30cl | 水 |
| 适量 | 孜然、辣椒粉 |

## 步骤 | Steps

1. 用粗盐按摩鲑鱼，然后放入冰箱腌制3小时。
2. 将甜椒切成条，在橄榄油中煎。
3. 甜椒变软金黄后，加入压扁的蒜瓣、柠檬皮、辣椒粉、橄榄油和水。
4. 小火炖10分钟，加入一束香菜，再炖10分钟。备用。
5. 在烤盘上，用一半甜椒和一半香菜铺底。放上鲑鱼块，然后用剩余的甜椒和香菜覆盖。
6. 用锡纸覆盖，放入210°C烤箱烤30分钟。

## 小贴士 | Tips

- 腌制过程可以让鱼肉更紧实入味。
- 烤制时锅内应该保持微沸状态。
- 搭配米饭或烤蔬菜食用。
