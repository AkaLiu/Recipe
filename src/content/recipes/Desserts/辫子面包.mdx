## 食材

| 用量 | 食材 |
|------|------|
| 500g | T55面粉 |
| 20g | 面包酵母 |
| 20cl | 水 |
| 5cl | 葵花籽油 |
| 50g | 糖 |
| 6g | 盐 |
| 2个 | 鸡蛋 |

## 步骤

1. 在厨师机碗中，将酵母放入温水中，加入油、一个鸡蛋和一点糖。
2. 密封碗口，等待酵母激活：得到一个棕色、有气泡、有香味的糊状物。
3. 筛入面粉、糖和盐。
4. 使用厨师机的揉面钩，2档，慢慢加入面粉、糖和盐。2档揉10分钟，然后4档揉5分钟。
5. 将面团从揉面钩上取下，密封碗口，在30°C下发酵2小时。
6. 面团发酵后，切成4个小面团，在烤盘上编成辫子。密封盖好，在30°C下再发酵2小时。
7. 打一个鸡蛋，涂在面包上，然后放入180°C烤箱烤20分钟。

## 小贴士

- **注意**：温度超过30°C酵母会死亡，水不能太烫。
- 这是犹太安息日面包，传统是6股辫子。
- 可以撒上芝麻装饰。
