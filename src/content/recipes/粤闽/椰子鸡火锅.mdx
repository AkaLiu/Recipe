## 食材

### 调料

| 用量 | 食材 |
|------|------|
| 4个 | 小青桔 |
| 适量 | 鱼露 |
| 适量 | 蒜末 |
| 3根 | 小米辣 |
| 适量 | 葱花和香菜 |

### 锅底

| 用量 | 食材 |
|------|------|
| 2个/1L | 椰青/椰子水 |

### 涮物

| 用量 | 食材 |
|------|------|
| 适量 | 文昌鸡 |
| 适量 | 千页豆腐 |
| 适量 | 豆皮 |
| 适量 | 金针菇 |
| 适量 | 西洋菜 |
| 适量 | 其他喜欢的食材 |

## 步骤

1. 准备调料：将小青桔挤出汁，加入鱼露、蒜末、小米辣、葱花和香菜混合均匀。
2. 锅底：将两个1L的椰青打开，倒入椰子水和椰肉。
3. 水开后，先涮鸡肉片。
4. 之后依次涮入其他食材。

## 小贴士

- 文昌鸡是海南特产，皮脆肉嫩。
- 椰子水清甜，鸡肉鲜嫩。
- 蘸料可以根据个人口味调整。
