## 食材 | Ingredients

| 用量 | 食材 |
|------|------|
| 500g | 嫩豆腐（北豆腐） |
| 150g | 猪肉末 |
| 1 tbsp | 郫县豆瓣酱 |
| 1 tbsp | 豆豉 |
| 2瓣 | 蒜末 |
| 1块 | 姜末 |
| 2根 | 葱花 |
| 1 tsp | 花椒粉 |
| 2 tbsp | 植物油 |
| 1 tbsp | 水淀粉 |
| 200ml | 鸡汤 |

## 步骤 | Steps

1. 热锅凉油，加入郫县豆瓣酱和豆豉，小火炒出红油。

2. 加入猪肉末煸炒至变色，撇去多余油脂。

3. 加入蒜末、姜末和一半的葱花，炒香。

4. 倒入鸡汤煮沸，轻轻放入豆腐块。

5. 小火炖煮5-7分钟，轻轻晃动锅子（不要用铲子翻动，以免豆腐碎掉）。

6. 加入水淀粉勾芡，撒上花椒粉。

7. 撒上剩余葱花，即可出锅。

## 小贴士 | Tips

- 使用嫩豆腐（北豆腐）口感更好，但要注意轻拿轻放
- 郫县豆瓣酱是灵魂，不要省略
- 花椒粉最后撒上，麻味更浓郁
- 可以根据口味调整辣度
